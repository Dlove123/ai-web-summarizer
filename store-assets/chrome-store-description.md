# Chrome Web Store Listing

## 短描述（132字符）
一键生成网页内容AI摘要，支持导出到Notion/飞书。完全免费，无限次使用。

## 详细描述

📝 **AI网页摘要助手** - 让信息获取更高效

在信息爆炸的时代，快速获取网页核心内容变得至关重要。AI网页摘要助手使用先进的大语言模型，一键生成网页内容的智能摘要，帮助你：

✨ **核心功能**

• **一键摘要** - 选中网页任意文字，按Alt+S瞬间生成AI摘要
• **整页提取** - 自动识别文章正文，智能过滤广告和无关内容
• **三种模式** - 精简（2-3句）、详细（核心观点+论据）、要点（5-7条）
• **一键导出** - 复制到剪贴板，或导出到Notion/飞书文档
• **历史记录** - 自动保存摘要历史，支持云端同步
• **完全免费** - 无订阅费，无使用次数限制

🚀 **使用场景**

• 研究人员快速浏览大量论文
• 学生整理学习资料
• 内容创作者收集素材
• 知识工作者高效阅读
• 任何需要快速获取信息的人

🔒 **隐私保护**

• 所有处理在本地浏览器完成
• API密钥仅存储在你的浏览器中
• 不收集任何个人数据
• 开源代码，透明可审查

📋 **API配置说明**

本插件使用通义千问或DeepSeek API生成摘要：

1. **通义千问**（推荐）
   - 免费额度：100万Token/月
   - 获取：https://dashscope.aliyun.com/api-key
   
2. **DeepSeek**
   - 免费额度：5000万Token/月
   - 获取：https://platform.deepseek.com

配置方法：
- 点击插件图标 → 设置 → 选择API提供商 → 输入API Key → 保存

💡 **使用技巧**

• 快捷键 Alt+S 快速打开插件
• 右键选中文字 → "AI摘要选中内容"
• 右键页面 → "AI摘要整个页面"
• 支持批量摘要，历史记录永久保存
